Sync Between Devices
Move your Navigator memory between phone and laptop, or receive a Philosophical Profile from the PRF-22 app. Uses a one-time 6-character code valid for 10 minutes. Nothing is stored on the server — data is deleted the moment it is received.
How this works

When you send, your Navigator memory is compressed and held in server memory for up to 10 minutes. A 6-character code (or QR code) lets your other device retrieve it. The data is deleted immediately when received — it cannot be fetched twice. If the timer expires before you receive, nothing is retained on the server. Use this for phone ↔ laptop transfers and for importing your PRF-22 Philosophical Profile into the Navigator.

What gets sent: your complete Navigator memory — the same as Export JSON. Twin conversations, Coherence Check history, Field Scans, Move Cards, PRF profile, Guide Notes, and Living Threads.

What is this?

A Field Scan is a situational map. It helps the Twin understand the environments acting on you: school, work, family, research, finances, community, or any field where agency may route or get suppressed.

Show me the lived version

You already know what a field feels like. A classroom, workplace, family system, or office can make some actions feel easy, risky, invisible, rewarded, or costly. A Field Scan helps you make that invisible shaping visible.

The suppression gap is the feeling of having something to contribute that the environment gives no good route to express.

Your Field Map

Your Twin learns who you are from conversation, what you create from Document Capsules, and what is acting on you from Field Scans. Several scans help reveal the combined situation-context in which action becomes possible, costly, urgent, meaningful, or blocked.

Coherence Check

Before choosing a move, pause and test your map. This is not a school exercise or a diagnosis. It is a way to notice whether your interpretation, evidence, values, feelings, and next action are working together or pulling apart.

How this connects to critical thinking

These everyday questions practice traditional critical thinking in coherence language. “What am I taking for granted?” is an assumption check. “What actually supports my interpretation?” is an evidence check. “What else might be going on?” is an alternative-explanation check. “What part am I leaving out?” is a completeness check. “What move would reduce fragmentation?” turns reasoning into practical judgment.

What is a BioBond?

A BioBond is a person, practice, place, memory, or community that helps you stay coherent under pressure. Many cases of ethical resilience suggest that principled action is easier to sustain when people have concrete supports that help them remain themselves.

What Navigator does

Navigator helps a user talk with a Digital Twin, preserve memory through Journal entries and Document Capsules, map life-fields through Field Scans, assemble the combined context, generate possible Move Cards, compare moves across fields, and create a small Current Move Plan for deliberation.

What Navigator does not do

  • It does not diagnose, treat, grade, command, or replace professional advice.
  • It does not decide who the user is or what the user must do.
  • It does not store raw uploaded documents after a Document Capsule is created.
  • It does not cache backend AI calls, API responses, or raw uploaded document text.

Privacy and security posture

Local-first by design: conversation, Journal entries, Coherence Check readings, PRF-22 profile summaries, Document Capsules, Field Scans, Context Assembly, Move Cards, comparison state, Current Move Plans, and Guide Notes are stored in this browser unless the user exports or restores them.

Backend transparency: backend AI calls occur only when the user asks the Twin to interpret, summarize, deliberate, create a capsule, or generate an AI-assisted response. Those requests require sending selected prompt context to the configured backend service for processing. PRF-22 import stores the raw JSON locally and sends only a short profile summary when Twin Context is turned on. Users should avoid uploading highly sensitive material unless they understand that selected content may be sent for AI processing.

User control: users can export JSON, export TXT, import/restore JSON, delete saved items, and clear local memory. Document uploads create reviewable capsules; the app saves only the capsule the user approves, not the raw uploaded document text.

Help / Tutorial
A quick guide to using Navigator without needing the theory first.

Quick Start

  1. Click Start / Resume Twin.
  2. Reply briefly and honestly. The Twin should ask one question at a time.
  3. After a few replies, expand the Twin Journal to see what the Twin has cautiously learned.
  4. Upload a document if you want the Twin to understand a project or research style.
  5. Run a Field Scan if you want to map an environment such as school, work, family, finances, or research.
  6. Generate Move Cards from saved Field Scans, then use Cross-Field Move Comparison to compare two to four possible moves.
  7. Set one Current Move when you are ready to deliberate. Current does not mean final; it means “the move I am now considering.”
  8. Create a Current Move Plan that names the smallest first action, support needed, risk to watch, non-sacrifice point, and check-back moment.
  9. Export JSON before changing browsers, devices, or clearing memory.

Three ways to use Navigator

  • Self-understanding: talk with the Twin and review the Twin Journal.
  • Research or creative work: upload documents and build capsules that show themes, style, and open questions.
  • Life navigation: run Field Scans to understand which environments support or suppress agency.
  • Action deliberation: compare saved Move Cards across fields before choosing what to treat as the current next move.

Example use case

A student feels stuck in a course. They talk with the Twin and notice they care about meaningful work, not just grades. They run a Field Scan and see that effort feels absorbed by compliance tasks. They identify one BioBond: a study partner who helps them think clearly. Their next move is not perfect, but it preserves agency.

Plain-language glossary

Twin JournalBrief, editable learnings the Twin gathers from conversation.
Document CapsuleA short approved memory made from a document, not the raw file itself.
Field ScanA map of an environment and how it shapes what you can express or do.
Suppression GapThe gap between what you could contribute under supportive conditions and what you feel able to express now.
BioBondA person, practice, place, or structure that helps your agency stay alive under pressure.
Agency-preserving next moveThe next survivable, coherent action that keeps authorship and possibility open.
Decision lensA temporary way of viewing moves, such as urgency, future agency, support-readiness, or reversibility. It is a perspective, not an order.
Current Move PlanA small, revisable action plan for the move the user is presently considering. It names why now, first action, support, risk, non-sacrifice, and check-back.
Guide NoteA saved output from a guided practice or workbook import. Older exports may still call these coach notes for compatibility.
